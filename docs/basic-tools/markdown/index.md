## 简介

&emsp;&emsp;Markdown是{==一种轻量级标记语言==}，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与HTML混编，也可导出HTML、PDF以及本身的md格式文件[^1]。
